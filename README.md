# stressor
project for hack club maccondo and AP bio, basically is just loud, bright and annoying. Randomly buzzes piezo buzzers and also rings a bell for maximum annoyance.

Materials: 
1x arduino (any model should work)
2x piezo buzzers 
1x servo motor (i used an mg90 servo but you could use whatever small servo, you just might have to tweak the code)
1x bell (any bell can work, youll just have to figure out how to attatch it to the servo horn, I 3d printed an adapter)



Wiring:
 *   Buzzer 1:  positive → Pin 8,  negative → GND
 *   Buzzer 2:  positive → Pin 9,  negative → GND
 *   Servo:     signal   → Pin 10, power → 5V, GND → GND

the adapter i used for attatching the servo to the bell is included in the repository, however you will likely need to print your own
